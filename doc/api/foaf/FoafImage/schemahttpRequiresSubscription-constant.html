IriTerm const schemahttpRequiresSubscription

requiresSubscription from schemahttp vocabulary [Expects: http://schema.org/Boolean, http://schema.org/MediaSubscription]

Indicates if use of the media require a subscription (either paid or free). Allowed values are true or false (note that an earlier version had 'yes', 'no').

Can be used on: http://schema.org/ActionAccessSpecification, http://schema.org/MediaObject
